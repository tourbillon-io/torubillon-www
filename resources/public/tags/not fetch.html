Scheduled Tasks

Schedule one-time or repeating tasks. Use a supported client library or access our HTTP service directly to simply schedule an event. Trigger immediate events, schedule for a specific time, or use a cron-like format for more flexible recurring tasks.

Supported Task Types:

  • Email
  • SMS
  • Web Hook

For more details on scheduled tasks, see our task use cases.

Workflows

Let tourbillon manage your workflows. Create jobs with flexible state transitions. Query for the current state of a job, or react to state transitions by sending email/sms or callling your own webhooks.

For more details on workflows, see our workflow use cases.
